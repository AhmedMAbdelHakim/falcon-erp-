# دليل الإقفال الشهري

1. ابدأ دورة الإقفال للفترة المطلوبة.
2. عالج الأحداث غير المرحلة والمخزون السالب والموافقات المفتوحة.
3. طابق المحافظ وحسابات العملاء والموردين والشحن والرواتب والشركاء.
4. أثبت كل بند في القائمة مع القيمة المتوقعة والفعلية والدليل.
5. نفذ التحقق؛ يجب أن يتساوى ميزان المراجعة وتكتمل البنود المانعة.
6. احصل على الموافقة المنفصلة ثم أغلق الفترة.

بعد الإغلاق لا تعدّل المصدر أو القيد. الخطأ يصحح بعكس أو قيد تعديل في فترة مفتوحة مع مرجع الفترة المتأثرة. إعادة الفتح استثناء يحتاج طلبًا وموافقة وسببًا موثقًا.
