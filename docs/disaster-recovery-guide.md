# Disaster Recovery Guide

## Triage

Declare an incident, stop affected sensitive commands, preserve logs/audit data and identify the last trusted backup and migration head. Do not improvise direct financial edits.

## Recovery

Restore into isolation, apply reviewed forward migrations, restore object bytes, reconcile counts and accounting controls, then run security and workflow tests. Promote only with incident-owner, finance and technical approval.

The local measured database restore time is 5.52 seconds for a 1.92 MB synthetic dump. It is not a production RTO. Production RPO/RTO must be approved after staging tests at representative volume.
