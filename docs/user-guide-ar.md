# دليل المستخدم

سجّل الدخول بحساب العمل. تعرض القائمة الوحدات التي يسمح بها دورك فقط. استخدم البحث والتصفية داخل كل شاشة، واضغط **تحديث** لجلب الحالة المعتمدة.

عند تنفيذ إجراء، افتح زر الإجراء، أدخل المعرفات والقيم من السجل المعتمد، ثم راجعها قبل **تنفيذ**. يمنع النظام التكرار بمفتاح مستقل ويعرض معرف تتبع بعد النجاح. لا تعتبر إيداع العميل إيرادًا، ولا تعدّل الأرقام خارج الأوامر المعتمدة.

في الهاتف افتح القائمة من رمز الخطوط. يمكن تغيير المظهر من الشريط العلوي أو الإعدادات. عند ظهور رفض صلاحية تواصل مع المسؤول؛ لا تعِد المحاولة بحساب آخر لمشاركة البيانات.
