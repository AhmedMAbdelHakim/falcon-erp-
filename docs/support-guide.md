# Support Guide

Capture timestamp, user, role, organization, route, browser, correlation ID, expected/actual behavior and a redacted screenshot. Never request passwords, service keys or real financial exports in ordinary tickets.

Escalate immediately for suspected cross-organization access, journal imbalance, duplicate posting, closed-period mutation, missing audit event or backup failure. Disable only the affected sensitive capability through the approved incident owner; preserve evidence and do not directly edit financial data.
