# Warehouse Guide

Warehouse/operations users receive and QC print batches, manage eligible inventory events, create/update shipments and record returns through approved commands. Verify item, quantity, batch and organization before execution. Retrying the same command uses the same idempotency key.

Never correct stock with direct table edits. Use the approved reversal/correction workflow and retain physical-count evidence. Escalate negative stock, duplicate movement or state mismatch immediately.
