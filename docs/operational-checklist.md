# Operational Checklist

## Daily

- Review failed authentication, denied sensitive commands and audit anomalies.
- Review unallocated payments, order exceptions, unreceived print batches, inventory anomalies and unsettled courier activity.
- Confirm backup completion and alert delivery.
- Never edit posted financial rows; use approved reversal commands.

## Monthly

- Reconcile wallets, customers, suppliers, courier balances, payroll and inventory.
- Run close readiness, resolve blockers, obtain approvals, execute close once and archive evidence.
- Compare reports to ledger control accounts and record sign-off.

## Incident

- Disable affected sensitive commands, preserve logs/correlation IDs and notify the named owner.
- Restore only into an isolated environment until reconciliation proves correctness.
- Record decisions, timestamps, scope and recovery evidence in the incident log.
