# دليل الطوارئ

عند اشتباه تسريب أو خطأ مالي: أوقف الإجراء الحساس المتأثر، لا تحذف السجلات، احفظ معرفات التتبع والأوقات، وأبلغ مالك الحادث والمراجع المالي.

للمفاتيح: ألغِ المفتاح، دوّره، راجع السجل والحزم، ثم أعد النشر بموافقة. لعدم الاتزان أو خطأ الإقفال: امنع الترحيل، لا تفتح الفترة يدويًا، وشغّل تحقيقًا ومعالجة أمامية معتمدة. لفشل النسخ الاحتياطي: أوقف أي قطع إنتاجي حتى نجاح نسخة واختبار استعادة.

لا تستخدم `git reset --hard` أو إعادة ضبط قاعدة بعيدة أو تعديل migration مطبق كإجراء طوارئ.
